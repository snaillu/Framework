类选择器

类选择器通常的写法如下：

        .item{margin:0 auto;text-align:center;}

        .parent .child{height:50px;position:absolute;left:0;right:0;border:1px solid orange;}

        .main-left,.main-right{width:50%;height:100%;float:left;}

示例

示例1：

        


        给wrap添加样式：

        .wrap{border:1px solid orange;float:right;}

示例2：

        


        给wrap下面的inner添加样式：

        .wrap .inner{padding:2px;margin:10px;}
