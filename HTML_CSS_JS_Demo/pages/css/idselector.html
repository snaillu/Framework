ID选择器

ID选择器可以为标有特定ID的标签元素指定样式。根据标签的ID来选择元素，具有唯一性。

ID选择器通常的写法如下

        #filelist{padding:5px 2px 2px 2px;color:red;}

        #item a{color:white;font-size:1.5rem;}

        #index1,#index2{margin-left:20px;color:red;}

注意：根据元素ID来选择元素，具有唯一性，这意味着同一id在同一文档页面中只能出现一次。尽管你会发现随便 把几个标签都定义相同的ID，css选择器还是会把这些元素都选中应用样式。但出于编程的习惯，相同的id不要再页面中出现两次（JS中只选中第一个元素）。
